# Retail Sales Data Pipeline using Databricks

Built an end-to-end Medallion Architecture pipeline using:

- Databricks
- PySpark
- Delta Lake
- Unity Catalog

## Layers

### Bronze
- Orders CSV
- Products JSON
- Customers Source

### Silver
- Data Quality Validation
- Referential Integrity Checks
- Advanced Reject Framework

### Gold
- Order Fact Table
- City Sales Summary
- Category Sales Summary

## Tech Stack

- Databricks
- PySpark
- Delta Lake
- Python
- SQL
